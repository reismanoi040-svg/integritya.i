# integritya.i
O **IntegrityIA** é uma plataforma de IA que auxilia empresas na gestão de riscos psicossociais (NR-1). Oferece check-ins e relatos anônimos para colaboradores, inteligência de dados e planos preventivos para o RH, e relatórios consolidados para proprietários. Garanta saúde mental, segurança psicológica e conformidade legal em um só lugar.
